	with Twitch, jump over obstacles, compete on the leaderboard - Up to 4 players can race in
	real-time with identical obstacle layouts
</Callout>

FangDash is a fast-paced multiplayer endless runner built for Twitch streamers and their communities. Race as a **wolf**, dodge obstacles, and compete for the highest score.

## What is FangDash?

Your wolf runs forward automatically. Jump over barriers, dodge hazards, and reach the top of the leaderboard. In multiplayer, you and your Twitch chat race the same course at the same time.

## Features

| Feature                | Details                                                         |
| ---------------------- | --------------------------------------------------------------- |
| **Endless Runner**     | Jump, dodge, and dash through procedurally generated courses    |
| **Multiplayer Racing** | Race up to 4 players in real-time                               |
| **Twitch Integration** | Sign in with Twitch and race your chat                          |
| **Wolf Skins**         | 6 skins (1 default + 5 unlockable) across 5 rarity tiers        |
| **Achievements**       | 16 achievements across 5 categories                             |
| **Level System**       | Earn XP from scores and race placements to level up             |
| **Leaderboards**       | Filter by time period and difficulty with anti-cheat validation |
| **Difficulty Scaling** | Speed ramps from 300 to 800 px/s as you run further             |
| **Admin Panel**        | Manage players, moderate scores, and view race history          |
