FangDash includes a hidden debug menu for developers and admins.

## Accessing the Debug Menu

Press **Ctrl+Shift+D** during gameplay to toggle the debug panel. Access is restricted to users with `dev` or `admin` roles.

## Tabs

### Stats

Real-time game statistics:

- Current speed, score, and distance
- Frame rate and delta time
- Active difficulty tier

### Constants

View current game constants:

- Physics values (gravity, jump velocity)
- Speed and scoring multipliers
- Obstacle spawn rates

### Cheats

Developer shortcuts for testing:

- Set score or distance to specific values
- Toggle invincibility
- Skip to a specific difficulty tier
- Force-trigger achievements
- **Unlock All Skins** — instantly unlocks every skin for the current player (calls `admin.unlockAllSkins`)

## Visual Style

The debug menu uses a **CRT/retro terminal aesthetic** with scan lines and a green-on-black color scheme. This keeps it visually distinct from the main game UI.
