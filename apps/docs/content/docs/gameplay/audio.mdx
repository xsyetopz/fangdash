<Callout type="info">
	**TL;DR** - 3 BGM tracks switch automatically based on where you are (menu, solo, race) - 9 sound
	effects covering all major game events - Volume and mute settings are saved to `localStorage` and
	persist across sessions
</Callout>

FangDash features a full audio system with background music and sound effects.

## Background Music

Three **BGM tracks** play depending on where you are:

| Track    | When it plays                |
| -------- | ---------------------------- |
| Menu BGM | Main menu and lobby screens  |
| Game BGM | Solo endless runner gameplay |
| Race BGM | Multiplayer race gameplay    |

Music **switches automatically** when transitioning between screens — the current track stops and the new one starts.

## Sound Effects

| Sound       | Trigger                          |
| ----------- | -------------------------------- |
| Jump        | Player jumps                     |
| Double Jump | Player double-jumps              |
| Hit         | Player collides with an obstacle |
| Game Over   | Run ends                         |
| Milestone   | Distance milestone reached       |
| Countdown   | Race countdown ticks             |
| Achievement | Achievement unlocked             |
| Skin Equip  | New skin equipped                |
| Victory     | Race won                         |

## Volume Controls

- **Volume** — A single master volume slider controls all audio (BGM and SFX). Stored in `localStorage` as `fangdash_volume`.
- **Mute** — A toggle to mute/unmute all audio. Stored in `localStorage` as `fangdash_muted`.

Your preferences persist across sessions. Set once, forget about it.
