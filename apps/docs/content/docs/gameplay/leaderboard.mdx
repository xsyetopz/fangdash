	and bronze rank badges
</Callout>

The leaderboard shows the best scores across all FangDash players. Visit it anytime at [/leaderboard](/leaderboard).

## Time Period Filters

Switch between three time windows:

| Filter       | Shows                          |
| ------------ | ------------------------------ |
| **Daily**    | Best scores from the last 24h  |
| **Weekly**   | Best scores from the last 7d   |
| **All Time** | The highest scores ever posted |

## Difficulty Filtering

You can filter the leaderboard by **difficulty level**. Each difficulty tier is reached at a specific distance during gameplay:

| Difficulty    | Distance Threshold |
| ------------- | ------------------ |
| **Easy**      | 0m                 |
| **Medium**    | 500m               |
| **Hard**      | 1,500m             |
| **Insane**    | 3,000m             |
| **Nightmare** | 5,000m+            |

Scores are saved with the difficulty level they were achieved at. This lets you compare apples to apples.

## Personal Best

The leaderboard shows **one entry per player** — your personal best score for the selected filters. If you beat your own high score, your entry updates automatically.

## Rank Badges

The top 3 players get special badges:

| Rank | Badge  |
| ---- | ------ |
| 1st  | Gold   |
| 2nd  | Silver |
| 3rd  | Bronze |

## Anti-Cheat

All scores are **server-validated** before appearing on the leaderboard. Scores that exceed the theoretical maximum for a given run duration and obstacle count are automatically rejected. See [Scoring](/docs/gameplay/scoring) for details.
