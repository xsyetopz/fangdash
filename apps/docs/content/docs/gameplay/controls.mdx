<Callout type="info">
**TL;DR**
- Jump with Space, Up Arrow, or W on keyboard
- Tap anywhere on mobile
- You can **double jump** — press again while airborne for a second jump
</Callout>

FangDash supports keyboard and touch controls.

## Keyboard

| Key | Action |
| --- | --- |
| Space | Jump |
| Up Arrow | Jump |
| W | Jump |

## Touch / Mobile

Tap anywhere on the game screen to jump.

## Double Jump

Press jump **again while airborne** to perform a **double jump**. You get up to 2 jumps per airborne phase. Use it to clear taller obstacles or recover from a mistimed first jump.

## Controller

Controller support is planned for a future update.
