<Callout type="info">
	**TL;DR** - 16 achievements across 5 categories: Score, Distance, Games Played, Skill, Social -
	Three achievements unlock exclusive wolf skins as rewards - Progress is tracked automatically — no
	manual claiming needed
</Callout>

FangDash has **16 achievements** across 5 categories. Check your progress anytime at [/achievements](/achievements).

## Score

Earn these by hitting **score milestones** in a single run.

|     | Achievement   | Description                                    | Condition                  |
| --- | ------------- | ---------------------------------------------- | -------------------------- |
| 🐺  | First Fang    | Score 100 points in a single run               | Single run score >= 100    |
| ⚡  | Sharp Fangs   | Score 1,000 points in a single run             | Single run score >= 1,000  |
| 👑  | Apex Predator | Score 10,000 points in a single run            | Single run score >= 10,000 |
| 💰  | Score Hoarder | Accumulate 50,000 total points across all runs | Total score >= 50,000      |

## Distance

Earn these by running **long distances**.

|     | Achievement     | Description                             | Condition                     |
| --- | --------------- | --------------------------------------- | ----------------------------- |
| 🐾  | First Steps     | Run 500 meters in a single run          | Single run distance >= 500m   |
| 🏃  | Marathon Runner | Run 5,000 meters in a single run        | Single run distance >= 5,000m |
| 🌍  | World Traveler  | Run 50,000 total meters across all runs | Total distance >= 50,000m     |

> **Skin Reward:** 🏃 Marathon Runner unlocks the **Fire Wolf** skin.

## Games Played

Earn these by **playing lots of games**.

|     | Achievement | Description          | Condition        |
| --- | ----------- | -------------------- | ---------------- |
| 🐶  | Pup         | Play your first game | 1 game played    |
| 🐺  | Pack Member | Play 25 games        | 25 games played  |
| 🌙  | Lone Wolf   | Play 100 games       | 100 games played |

## Skill

Earn these by demonstrating **skillful play**.

|     | Achievement     | Description                                    | Condition                                        |
| --- | --------------- | ---------------------------------------------- | ------------------------------------------------ |
| 🔥  | Obstacle Dodger | Clear 100 obstacles total                      | 100 obstacles cleared                            |
| ⭐  | Obstacle Master | Clear 1,000 obstacles total                    | 1,000 obstacles cleared                          |
| 💎  | Perfect Dash    | Run 1,000 meters without hitting any obstacles | 1,000m clean run (tracked via `longestCleanRun`) |

> **Skin Reward:** ⭐ Obstacle Master unlocks the **Storm Wolf** skin.

## Social

Earn these by **racing with others**.

|     | Achievement   | Description              | Condition          |
| --- | ------------- | ------------------------ | ------------------ |
| 🏁  | First Race    | Complete your first race | 1 race completed   |
| 🏆  | Champion      | Win 10 races             | 10 races won       |
| 🎖️  | Veteran Racer | Complete 50 races        | 50 races completed |

> **Skin Reward:** 🏆 Champion unlocks the **MrDemonWolf** skin.
