FangDash has 16 achievements across 5 categories.

## Categories

### Score

Earn these by hitting score milestones in a single run.

- **First Fang** — Score 100 points in a single run
- **Century Runner** — Score 500 points in a single run
- **Score Master** — Score 1,000 points in a single run
- **Legendary Run** — Score 5,000 points in a single run

### Distance

Earn these by running long distances.

- **Getting Started** — Run 1,000 meters total
- **Marathon Wolf** — Run 10,000 meters total
- **Cross Country** — Run 50,000 meters total

### Games

Earn these by playing lots of games.

- **Newcomer** — Play 5 games
- **Regular** — Play 25 games
- **Dedicated** — Play 100 games
- **Obsessed** — Play 500 games

### Skill

Earn these by demonstrating skillful play.

- **Close Call** — Narrowly dodge an obstacle
- **Perfect Run** — Complete a run without any close calls

### Social

Earn these by playing with others.

- **First Race** — Complete your first multiplayer race
- **Pack Leader** — Win 10 multiplayer races
- **Alpha Wolf** — Win 50 multiplayer races

## Achievement Rewards

Some achievements unlock exclusive wolf skins. Check the [Skins](/docs/customization/skins) page to see which achievement unlocks each skin.
