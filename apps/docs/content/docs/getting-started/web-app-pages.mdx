<Callout type="info">
	**TL;DR** - 8 main pages: Home, Play, Race, Leaderboard, Achievements, Skins, Profile, Admin -
	Most pages require signing in with Twitch - Admin is restricted to admin/dev roles
</Callout>

Here's a quick tour of every page in the FangDash web app.

## Pages

| Page             | Route           | Description                                                             |
| ---------------- | --------------- | ----------------------------------------------------------------------- |
| **Home**         | `/`             | Landing page with a quick overview and sign-in prompt                   |
| **Play**         | `/play`         | Start a solo endless runner game                                        |
| **Race**         | `/race`         | Create or join a multiplayer race room                                  |
| **Leaderboard**  | `/leaderboard`  | Global rankings with time period and difficulty filters                 |
| **Achievements** | `/achievements` | View all 16 achievements and your unlock progress                       |
| **Skins**        | `/skins`        | Browse and equip wolf skins from the skin gallery                       |
| **Profile**      | `/profile`      | Your stats, level, recent scores, and equipped skin                     |
| **Admin**        | `/admin`        | Player management, score moderation, and race history (admin/dev roles) |

## Getting Around

The main navigation bar at the top links to Play, Race, Leaderboard, Achievements, and Skins. Your profile is accessible from the user menu in the top right corner.

## What Requires Sign-In

You need to **sign in with Twitch** to:

- Play games (scores are tied to your account)
- Join races
- Equip skins
- View your profile

The **Leaderboard** and **Achievements** list are viewable without signing in.
