FangDash lets you race against up to 3 other players in real-time. All players run through the same obstacle course, and the player with the highest score wins.

## How to Start a Race

1. Click **Multiplayer** from the main menu
2. Create a new room or join an existing one with a room code
3. Wait for other players to join (2-4 players)
4. When everyone is ready, the race begins

## Fair Play

Every player in a race faces the exact same obstacles. FangDash uses a seeded random number generator — the seed is shared when the race starts, so the obstacle layout is identical for everyone.

## What You See

During a race, you can see the other players' wolves running alongside yours as ghost players. You can't collide with them, but you can see their progress in real-time.

## Winning

The race continues until all players have hit an obstacle. The player with the highest score wins.
